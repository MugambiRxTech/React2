#Overview

This project is a React Single Page Application (SPA) built as a summative lab assessment.
It showcases projects for a creative agency or freelance developer using a modern, responsive interface.

#Features

View a list of projects on the landing page

Add new projects dynamically using a form

Search and filter projects by category

View project details using client-side routing

Delete projects

Persist data using localStorage

Fully responsive design

#Technologies

React (Vite)

React Router DOM

JavaScript (ES6)

CSS

Local Storage API
